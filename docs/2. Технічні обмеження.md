# Технічні обмеження

- Реєстрація користувачів за email/password
- Ролі та права доступу
- Підтримка декількох GA4 property
- Real-time аналітика
- Складні або інтерактивні графіки
- Власні події GA4
- Побудова власних ML-моделей
- Збереження історії звітів
- Реалізація повинна використовувати лише офіційні API Google
- Проєкт реалізується як MVP без enterprise-функціональності
- Пріоритет — стабільність та коректність роботи, а не кількість функцій
